# 🧠 AI News Dashboard

A full-stack AI-powered news dashboard built using **FastAPI, React, and SQLite**.

---

## 🚀 Features

- 📰 Fetch latest AI news
- ⭐ Add news to favorites
- 📂 View saved favorites
- 🔄 Real-time API integration
- 🌐 Full-stack deployment

---

## 🛠️ Tech Stack

### Backend:
- FastAPI
- SQLAlchemy
- SQLite

### Frontend:
- React.js
- Axios / Fetch API

### Deployment:
- Render (Backend)
- GitHub (Code hosting)
